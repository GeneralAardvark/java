Overview

Define: java

This module manages Java.

Parameters:

$java_version:: The java version. Possible values at this time are 6 and 7. Defaults to the resource's title.

$java_default_version:: The java default version. Possible values at this time are 6 and 7. If different than undef it will be used to configure the alternative system. Defaults to undef.

Actions:

Declares all other defines in the java module needed for installing Java. Currently, these consists of java::install, and java::config. If hiera defines a value for the parameter java_default_version on the target node the command update-alternatives is issued to set the default java accordingly.

Requires: see Modulefile

Sample Usage:

java {'6': }

  • java_version (Any) (defaults to: $title)
  • java_default_version (Any) (defaults to: hiera('java::java_default_version', undef))

define java (
  $java_version         = $title,
  $java_default_version = hiera('java::java_default_version', undef)) {

  if $facts['os']['family'] == 'Debian' {
    include apt
  }
  java::install_config { $java_version:
    java_default_version => $java_default_version,
  }
}
